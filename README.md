🎙️ AI Audio Transcription App

A full-stack application that converts speech (audio files) into text using AI.
Built with Spring Boot (Backend) and React (Frontend), this app allows users to upload audio files and get accurate transcriptions instantly.

🚀 Features
🎧 Upload audio files (MP3/WAV)
🤖 AI-powered speech-to-text conversion
📄 Clean and structured transcription output
📋 Copy-to-clipboard functionality
⚡ Fast and responsive UI
🔐 Secure API integration using environment variables
🛠️ Tech Stack
🔹 Frontend
React.js
Tailwind CSS (for UI styling)
Axios (API calls)
🔹 Backend
Spring Boot
REST API
Multipart file handling
🔹 AI Integration
Google Gemini API (for transcription)
